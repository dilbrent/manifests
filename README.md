
gtaxl-rros-lineage-15.1
===========

To initialize your local repository use a command like this:
````bash
repo init -u https://github.com/followmsi/manifests.git -b oreo
````
Then you need to clone the local_manifest:
````bash
git clone https://github.com/followmsi/manifests.git
git checkout origin/gtaxl-rros-lineage-15.1
````

Pls copy the "local_manifests" folder into your "./repo" folder inside your build tree.

Then to sync up:
````bash
repo sync
````
Finally to build:
````bash
source build/envsetup.sh

breakfast gtaxlwifi (gtaxllte)

croot

brunch gtaxlwifi (gtaxllte)
````
